# Zen Diorama

Zen Diorama is a NeoForge mod for Minecraft 1.21.1 that builds miniature diorama worlds inside decorative frame blocks.

## Release

- Version: `1.0.0`
- Minecraft: `1.21.1`
- Loader: NeoForge `21.1.220`

## Features

- Diorama frame blocks and items
- A dedicated diorama dimension and return flow
- World map blocks for miniature terrain presentation
- Client rendering for miniature blocks, lighting, and sky effects
- Deterministic plot allocation and snapshot sampling logic

## Install

1. Install Minecraft `1.21.1` with NeoForge `21.1.220`.
2. Download the release jar for `Zen Diorama` version `1.0.0`.
3. Place the jar in your `mods` folder.
4. Launch the game.

## Build From Source

```powershell
.\gradlew.bat test
.\gradlew.bat build
```
